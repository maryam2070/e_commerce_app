# e-commerce-app (Online Store)
online application provides online shopping experience to users

## Features

- Auth
    - sign in
    - sign up
    - sign out
    - set password via email
- Product
    - show product categories (new,sale)
- Cart
    - add product to cart
    - remove from cart
    - check out
 
- Order
    - make order
    - cancel order
    - display order data
    - add shipping address and payment data 


## Technologies 
- Dart
- clean archeticture 
- firebase 
- flutter_bloc
- equatable 
- get_it for dependency injection 
- fluttertoast
- flutter_rating_bar
